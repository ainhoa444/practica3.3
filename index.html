Navegadores web

Un navegador web (en inglés, web browser) es un software, aplicaión o programa que permite el acceso a la Web, interpretando la información de distintos tipos de archivos y sitios web para que esos puedan ser vistos

La funcionalidad básica de un navegador web es permitir la visualización de documentos de texto, probablemente con recursos multimedia incrustados

Ejemplos de navegadores web

Safari:Safari es un navegador web de código cerrado desarrollado por Apple Inc. Está disponible para macOS, iPadOS e iOS

Mozilla Firefox es un navegador web libre y de código abieto desarrollado para Linux, Android, iOS, macOS, y Microsoft Windows coordinado por la Corporación Mozilla y la Fundación Mozilla

Google Chrome Coogle Chrome es una navegador web de código cerrado desarrollado por Google, aunque derivado de proyectos de código abierto
